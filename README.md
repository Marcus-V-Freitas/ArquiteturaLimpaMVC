# ArquiteturaLimpaMVC

* Principais Funcionalidades:

- [x] Arquitetura Limpa (separação em camadas)
- [x] Definições de entidades de dominío rico
- [x] Testes de Unidade com XUnit
- [x] Validações personalizadas de dominío
- [x] Endpoint Gráfico feito em MVC
- [x] Endpoint de consumo feito em WebApi Rest   
- [x] Padrão Repositório
- [x] Implementação CQRS (Commands, Handlers e Queries) com Mediator
- [x] Documentação Swagger
- [x] Mapeamento de DTOs com AutoMapper
- [x] Consumo de Dados feito via EFCore
- [x] Validações de DTOs com DataAnnotations 
- [x] Autenticações cookies (MVC) e JWT (API)

## Tecnologias Utilizadas

- [C#](https://dotnet.microsoft.com/download)
